# Built-in Roles

| Role | Read | Create | Delete | Assign Roles |
|------|------|--------|--------|--------------|
| Reader | Yes | No | No | No |
| Contributor | Yes | Yes | Yes | No |
| Owner | Yes | Yes | Yes | Yes |
| User Access Administrator | RBAC only | No | No | Yes |

## Lab
Switch Reader → Contributor → Owner and validate permissions.
